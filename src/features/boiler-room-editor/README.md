# Boiler Room Editor Features

Эта папка содержит UI-компоненты редактора котельной: панели настроек, каталог, редактор карточек оборудования, SVG-план, принципиальную схему, свойства, validation panel и export panel.

Feature-компоненты могут собирать пользовательский ввод и отображать данные, но не должны становиться местом для новой доменной логики. Если нужно изменить правила geometry, validation, routing или connection resolving, сначала ищите соответствующий модуль в `src/domain` или `src/infrastructure`.

Видимый UI-текст должен быть на русском языке. TypeScript identifiers остаются на английском.
