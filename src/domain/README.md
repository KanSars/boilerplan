# Domain Layer

`src/domain` содержит чистую доменную модель и правила приложения. Здесь не должно быть React, Redux, DOM, browser download API, CSS или зависимостей от `src/infrastructure`.

Кладите сюда типы и функции, которые отвечают на вопросы инженерной модели: что такое проект, помещение, оборудование, точка подключения, маршрут, проверка, профиль стандартов или промежуточная drawing model.

Все координаты и размеры в этом слое хранятся в миллиметрах. Пиксели и SVG-specific детали принадлежат UI или exporter layer.

Ключевое различие:

- `SystemConnection` — логическая связность.
- `PipingRoute` — физическая предварительная трасса.
- `CadDrawing` — модель чертежа для CAD/DXF.
- `Requirement` / `EvidenceLink` — документальное основание и связь с источниками.

Не смешивайте эти понятия в одном типе.
